1. Install Tampermonkey
Open Google Chrome
Go to the Chrome Web Store
Search: “Tampermonkey”
Click Add to Chrome
Click Add Extension
2. Turn it on
After installing, you’ll see the Tampermonkey icon (🐵) in the top right
If you don’t see it, click the puzzle piece icon and pin it
3. Install a script

You have 3 easy ways:

Method A (easiest)

Go to a script website (like a .user.js file)
Click Install
Tampermonkey will pop up → click Install again

Method B

Click the Tampermonkey icon
Choose Dashboard
Click “+” or “Create new script”
Paste the script code
Click Save

Method C

Drag and drop a .user.js file into Chrome
4. Make sure it works
Go to the website the script is meant for
If it’s active, Tampermonkey icon will show a number
Common fixes if it doesn’t work
Enable “Allow user scripts” in Tampermonkey settings
Turn on Developer Mode in Chrome extensions page (chrome://extensions)
Refresh the page after installing
